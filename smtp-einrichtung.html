G

Gmail einrichten

Google-Konto mit @gmail.com oder G Suite / Google Workspace

⏱ Ca. 5 Minuten · erfordert App-Passwort
1
2-Faktor-Authentifizierung aktivieren
Gmail erfordert zwingend die 2-Faktor-Authentifizierung, bevor ein App-Passwort erstellt werden kann. Falls Sie diese bereits aktiviert haben, gehen Sie direkt zu Schritt 2.
myaccount.google.com → Sicherheit → 2-Schritt-Verifizierung
→ Google Sicherheitseinstellungen öffnen
2
App-Passwort erstellen
Ein App-Passwort ist ein spezielles 16-stelliges Passwort nur für PROVA — es ist sicherer als Ihr normales Passwort und kann jederzeit separat widerrufen werden.
myaccount.google.com → Sicherheit → 2-Schritt-Verifizierung → App-Passwörter
Auf der App-Passwort Seite:
1. Wählen Sie bei "App auswählen" → Sonstige (benutzerdefinierter Name)
2. Geben Sie ein: PROVA Gutachten-System
3. Klicken Sie auf Erstellen
4. Google zeigt Ihnen ein 16-stelliges Passwort (z.B. abcd efgh ijkl mnop)
5. Kopieren Sie dieses Passwort — es wird nur einmal angezeigt
→ App-Passwörter direkt öffnen
3
In PROVA eintragen
Tragen Sie die folgenden Werte in die PROVA-Einstellungen ein:
SMTP-Serversmtp.gmail.com
Port587
E-Mail-Adresseihre.adresse@gmail.com
PasswortApp-Passwort aus Schritt 2
⚠️ Wichtig: Tragen Sie das App-Passwort ein — nicht Ihr normales Google-Passwort. Das App-Passwort enthält keine Leerzeichen, auch wenn Google es mit Leerzeichen anzeigt.

🔒 Warum ist das sicher?

Ihr normales Google-Passwort wird niemals in PROVA eingegeben. Das App-Passwort gilt ausschließlich für PROVA und kann unter myaccount.google.com → Sicherheit → App-Passwörter jederzeit widerrufen werden, ohne Ihr Google-Konto zu gefährden. PROVA speichert das Passwort nur in Ihrem Browser — niemals auf fremden Servern.

Bereit zum Einrichten?

Gehen Sie zurück zu den Einstellungen und wählen Sie Gmail aus.

G

GMX einrichten

GMX Mail mit @gmx.de, @gmx.net oder @gmx.at Adresse

⏱ Ca. 2 Minuten · normales Passwort reicht
1
POP3/IMAP und SMTP aktivieren
Bei GMX muss der externe E-Mail-Zugriff einmalig aktiviert werden. Dies ermöglicht PROVA, E-Mails in Ihrem Namen zu senden.
gmx.de → Einstellungen (⚙) → E-Mail → Mein Konto → POP3 & IMAP
Setzen Sie den Haken bei "POP3 und IMAP Zugriff aktivieren" und speichern Sie.
→ GMX Konto-Einstellungen öffnen
2
In PROVA eintragen
Tragen Sie die folgenden Werte ein. Als Passwort verwenden Sie Ihr normales GMX-Passwort.
SMTP-Servermail.gmx.net
Port587
E-Mail-Adresseihre.adresse@gmx.de
PasswortIhr GMX-Passwort

🔒 Sicherheitshinweis

Ihr Passwort wird ausschließlich lokal in Ihrem Browser gespeichert und niemals an PROVA-Server übertragen — es reist nur verschlüsselt (HTTPS) im Moment des Versands. Falls Sie ein sichereres Verfahren bevorzugen, können Sie bei GMX unter Einstellungen → Sicherheit ein separates App-Passwort erstellen.

Bereit?

GMX ist in 2 Minuten eingerichtet.

1&1

IONOS / 1&1 einrichten

IONOS Mail oder 1&1 Hostingpaket mit eigener Domain

⏱ Ca. 2 Minuten · direkt einsatzbereit
1
Keine Voreinrichtung nötig
IONOS und 1&1 unterstützen SMTP standardmäßig. Sie benötigen nur Ihre E-Mail-Adresse und Ihr E-Mail-Passwort (nicht das IONOS-Kontrollcenter-Passwort).
Wo finde ich mein E-Mail-Passwort?
Mein IONOS → E-Mail → E-Mail-Adressen → Ihre Adresse anklicken → Passwort anzeigen

Falls Sie das Passwort nicht mehr wissen, können Sie es im IONOS Kundencenter zurücksetzen.
→ IONOS SMTP-Einstellungen
2
In PROVA eintragen
SMTP-Serversmtp.ionos.de
Port587
E-Mail-Adressename@ihredomain.de
PasswortIhr E-Mail-Passwort
Für 1&1 Hosting (ältere Pakete) kann der Server smtp.1und1.de sein. Testen Sie beide.

Bereit?

IONOS ist der einfachste Setup.

Telekom / T-Online einrichten

T-Online E-Mail mit @t-online.de, @telekom.de oder @magenta.de

⏱ Ca. 3 Minuten · Passwort-Einstellung beachten
1
E-Mail-Passwort überprüfen
Bei Telekom gibt es ein separates E-Mail-Passwort, das sich vom Telekom-Login-Passwort unterscheiden kann. Prüfen Sie, ob Sie ein separates E-Mail-Passwort gesetzt haben.
telekom.de/kundencenter → Mein Konto → E-Mail-Dienste → Passwort
→ Telekom Hilfe
2
In PROVA eintragen
SMTP-Serversecuresmtp.t-online.de
Port587
E-Mail-Adressename@t-online.de
PasswortIhr E-Mail-Passwort
⚠️ Wichtig: Verwenden Sie securesmtp.t-online.de (mit "secure" am Anfang), nicht smtp.t-online.de.

Bereit?

Telekom in 3 Minuten eingerichtet.

Outlook / Hotmail einrichten

Microsoft-Konto mit @outlook.de, @hotmail.de oder @live.de

⏱ Ca. 3 Minuten · App-Passwort empfohlen
1
App-Passwort erstellen (empfohlen)
Für maximale Sicherheit empfehlen wir ein App-Passwort. Dafür muss die 2-Faktor-Authentifizierung aktiviert sein.
account.microsoft.com → Sicherheit → Erweiterte Sicherheitsoptionen → App-Kennwörter
→ Microsoft Sicherheitseinstellungen
Alternativ: Wenn Sie keine 2FA haben, funktioniert auch Ihr normales Microsoft-Passwort. Dafür muss unter account.microsoft.com → Sicherheit → "Weniger sichere Apps erlauben" aktiviert sein.
2
In PROVA eintragen
SMTP-Serversmtp-mail.outlook.com
Port587
E-Mail-Adressename@outlook.de
PasswortApp-Passwort oder normales Passwort

Bereit?

Outlook in wenigen Minuten eingerichtet.

⚙

Eigener Mail-Server

Eigene Domain mit professionellem Hosting (z.B. Hetzner, Strato, All-Inkl, eigener Server)

⏱ Daten beim Hoster nachschlagen
1
SMTP-Daten beim Hoster nachschlagen
Ihre SMTP-Zugangsdaten finden Sie im Kundencenter Ihres Hosting-Anbieters oder in der Willkommens-E-Mail. Suchen Sie nach "E-Mail einrichten", "SMTP-Server" oder "Outgoing Mail Server".
Typische SMTP-Einstellungen bei gängigen Hostern:

Strato: smtp.strato.de · Port 587
All-Inkl: w0123abc.kasserver.com · Port 587
Hetzner: mail.your-server.de · Port 587
Webgo: smtpausw.webgo.de · Port 587
DomainFactory: smtp.df.eu · Port 587
2
Manuell in PROVA eintragen
Wählen Sie in PROVA "Eigener Server" und tragen Sie die Daten Ihres Hosters ein. Der Port ist fast immer 587 (STARTTLS).
⚠️ Falls Port 587 nicht funktioniert, versuchen Sie Port 465 (SSL/TLS). Beides sind sichere Verbindungen.
3
Verbindung testen
Klicken Sie auf "Verbindung testen" in PROVA. Sie erhalten eine Test-Mail an Ihre eigene Adresse. Bei Fehler zeigt PROVA eine genaue Fehlermeldung mit Hinweis.

Bereit?

Eigener Server einrichten.
