# Floskeln-Seed-Daten (50 Floskeln für TEXTBAUSTEINE_CUSTOM)

> **Einspielung:** nachdem `laenge`-Feld (Single Select: kurz/mittel/lang) in TEXTBAUSTEINE_CUSTOM angelegt wurde.
>
> **Methode:** Entweder per CSV-Import in Airtable oder per Claude-Code-Script (siehe unten).
> Alle Floskeln: `laenge = "kurz"`, `sv_email` = deine Email als Vorlage, Marcel kann später `sv_email = "system@prova-systems.de"` setzen wenn globale Verfügbarkeit gewünscht.

## Einspiel-Script (Node.js) für Marcel/Claude Code

```javascript
// seed-floskeln.js
// Ausführen mit: AIRTABLE_PAT=xxx SV_EMAIL=xxx node seed-floskeln.js

const Airtable = require('airtable');
const base = new Airtable({apiKey: process.env.AIRTABLE_PAT}).base('appJ7bLlAHZoxENWE');

const FLOSKELN = [
  { kategorie: 'Ortstermin-Eröffnung', titel: 'OT-Eröffnung 1', text: 'Die Ortsbesichtigung fand am [DATUM] um [UHRZEIT] statt. Anwesend waren [TEILNEHMER].' },
  { kategorie: 'Ortstermin-Eröffnung', titel: 'OT-Eröffnung 2', text: 'Der Ortstermin wurde im Einvernehmen mit dem Auftraggeber am [DATUM] durchgeführt.' },
  // ... alle 50 aus obiger Liste ...
];

(async function() {
  for (var i = 0; i < FLOSKELN.length; i += 10) {
    var batch = FLOSKELN.slice(i, i + 10);
    var records = batch.map(function(f) {
      return {
        fields: {
          sv_email: process.env.SV_EMAIL,
          titel: f.titel,
          text: f.text,
          kategorie: f.kategorie,
          laenge: 'kurz',  // FELD MUSS EXISTIEREN!
          notiz: 'Seed-Floskel v1'
        }
      };
    });
    await base('TEXTBAUSTEINE_CUSTOM').create(records, { typecast: true });
    console.log('Batch ' + (i/10 + 1) + ' eingespielt');
  }
  console.log('✅ Alle 50 Floskeln eingespielt');
})();
```

## Workflow für Marcel

1. **Feld anlegen:** TEXTBAUSTEINE_CUSTOM → + Field → "laenge" → Single Select → Optionen: `kurz`, `mittel`, `lang` → Save
2. **Script ausführen** (nach git-Install + Claude Code läuft):
   ```bash
   cd prova-systems
   export AIRTABLE_PAT=xxx
   export SV_EMAIL=deine@email.de  
   node seed-floskeln.js
   ```
3. **Verifizieren:** in Airtable TEXTBAUSTEINE_CUSTOM → 50 neue Records mit laenge=kurz

**Alternative CSV-Import:** Alle 50 Zeilen als CSV mit Spalten `sv_email,titel,text,kategorie,laenge,notiz` und in Airtable-UI hochladen.
