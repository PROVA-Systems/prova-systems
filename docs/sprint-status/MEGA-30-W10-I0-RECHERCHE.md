## BLOCKER-RECHERCHE-PAKETE

### Blocker A — Einträge-System (≥10 Quellen)

1. **DSGVO Art. 5 c** — Datensparsamkeits-Grundsatz
2. **DSGVO Art. 32** — Sicherheit der Verarbeitung
3. **§ 257 HGB** — Aufbewahrungspflicht 10 Jahre

5. **JVEG § 8** — Vergütung des Sachverständigen
6. **JVEG § 12** — Stundensatz
7. **GDPdU** — Grundsätze zum Datenzugriff (Steuerprüfung)
8. **GoBD** — Grundsätze ordnungsmäßiger DV-gestützter Buchführungssysteme
9. **§ 10 IHK-SVO** — Akten-Führungs-Pflicht
10. **§ 407a Abs. 1 ZPO** — Eigenleistung-Dokumentation
11. **Bundesarchivgesetz** — Archiv-Standards (Best-Practice-Quelle)

**Schema-Decision:**
- Tabelle `eintraege` mit Spalten: id (UUID), schadensfall_id (FK), eintrag_typ (ENUM), datum, dauer_min (INT für JVEG), beschreibung_text (TEXT), anhang_files (JSONB), abrechenbar (BOOL DEFAULT TRUE), erstellt_von (UUID), erstellt_am (TIMESTAMPTZ), deleted_at (TIMESTAMPTZ NULL für Soft-Delete)
- ENUM `eintrag_typ`: ortstermin, telefonat, email, recherche, gutachten-arbeit, akteneinsicht, korrespondenz, sonstiges
- RLS: workspace-bezogen (JOIN über schadensfall.workspace_id)
- Index: schadensfall_id + datum DESC für Listenansicht

### Blocker B — Skizzen-Funktion (≥10 Quellen)

1. **W3C SVG 2.0 Spec** — Vector Graphics Standard
2. **DIN 18205** — Bauzeichnung-Format
3. **DIN 824** — Falten von Zeichnungen
4. **Excalidraw GitHub** — Open-Source Hand-drawn-Style Editor
5. **Fabric.js Documentation** — Canvas-basierte Zeichnungs-Lib
6. **TPF Pointer Events** — Touch+Stylus W3C Standard
7. **WAI-ARIA Authoring Practices** — Accessibility für interaktive Editoren
8. **§ 6 IHK-SVO** — Sachverständigen-Dokumentations-Pflicht
9. **DIN ISO 128** — Technische Zeichnung-Allgemeine-Grundlagen
10. **HTML5 Canvas API** — Native Zeichnungs-Backend
11. **iOS-Pencil API** — Stylus-Integration für iPad-Außendienst

**Tech-Decision (CTO-Vote):**
- **Excalidraw-Embed** als iframe für Browser-only Skizzen
- **Pro:** open-source, hand-drawn-Style, mobile-friendly, kein Custom-Maintenance
- **Contra:** iframe-Embedding, externe Lib-Abhängigkeit
- **Save-Format:** SVG-Export (Excalidraw → SVG via API), gespeichert als TEXT in Supabase
- **Welle-X-Alternative:** falls Excalidraw nicht ausreicht → Fabric.js Custom-Editor

**Schema-Decision:**
- Tabelle `skizzen`: id (UUID), schadensfall_id (FK), titel (TEXT), svg_content (TEXT), foto_referenz_id (FK optional), erstellt_von (UUID), erstellt_am (TIMESTAMPTZ)
- RLS: workspace-bezogen

### Blocker C — Fristen-System (≥10 Quellen)

1. **§ 225 ZPO** — Verfahrensfristen
2. **§ 411 Abs. 1 ZPO** — Frist zur Erstattung des Gutachtens
3. **§ 411 Abs. 4 ZPO** — Stellungnahmefrist
4. **DSGVO Art. 12 Abs. 3** — 1-Monats-Antwort-Frist
5. **§ 199 BGB** — Verjährung Beginn
6. **§ 195 BGB** — Regelverjährung 3 Jahre
7. **§ 638 BGB** — Verjährung Werkmängel-Ansprüche
8. **§ 4 Abs. 7 VOB/B** — Mängelbeseitigung-Frist
9. **§ 13 Abs. 5 VOB/B** — Mängelansprüche nach Abnahme
10. **BfDI Art. 12 DSGVO Leitfaden** — Antwort-Fristen
11. **§ 91a ZPO** — Beweissicherung-Fristen

**Schema-Decision:**
- Tabelle `fristen`: id (UUID), schadensfall_id (FK), frist_typ (ENUM), datum_soll, datum_ist, status (ENUM offen/erfuellt/verfallen), erinnerung_tage_vor INT[] DEFAULT '{14,7,3,1}', notiz, erstellt_am, deleted_at
- ENUM `frist_typ`: gericht / gutachten-erstellung / honorar / widerspruch / akteneinsicht / zeugen / parteien / ortstermin
- ENUM `frist_status`: offen / erfuellt / verfallen / verlaengert
- RLS: workspace-bezogen
- Cron-Trigger: tägliche Reminder-Mails 14/7/3/1 Tage vor `datum_soll`

**5 Pipeline-Definitionen:**
- Schadensgutachten-Pipeline: Auftrag → Ortstermin → Gutachten-Erstattung → Honorar
- Wertgutachten-Pipeline: Auftrag → Bewertungsstichtag → Gutachten → Honorar
- Bauabnahme-Pipeline: Auftrag → Termin → Mängelliste → Nachbesserung → Schlussabnahme
- Schiedsgutachten-Pipeline: Schiedsvereinbarung → Anhörung → Schiedsgutachten → Bindungs-Klausel
- Beweissicherung-Pipeline: Antrag → Termin → Beweismittel → Verjährungs-Hinweis

**8 Frist-Typen mit Default-Reminder-Pattern:**
- gericht: 14/7/3/1 Tage vor
- gutachten-erstattung: 21/14/7/3
- honorar: 30/14/7
- widerspruch: 7/3/1
- akteneinsicht: 14/7/3
- zeugen: 14/7
- parteien: 14/7/3
- ortstermin: 7/3/1
